## Install
This project has as a dependency the following the Backend:
#### ‚öôÔ∏è. [Backend](https://github.com/dimermichel/Ironhack-Project-3-API)
```sh
$ git clone https://github.com/dimermichel/Ironhack-Project-3-API.git
$ cd ironhack-project-3-api
$ npm install
$ npm start
```
You have to create and configure a .env file in the root folder with the following variables:

- PORT
- SESS_SECRET
- MONGODB_URI
- CLIENT_GITHUB_ID
- CLIENT_GITHUB_SECRET
- TRIPOSO_ID
- TRIPOSO_KEY
- WEATHERBIT_KEY
- MAPS_API_KEY
- BASE_URL
- FRONTEND_URL
- CLOUDINARY_NAME
- CLOUDINARY_KEY
- CLOUDINARY_SECRET

#### üñ•. [Frontend](https://github.com/dimermichel/Ironhack-Project-3-Frontend)

```sh
$ git clone https://github.com/dimermichel/Ironhack-Project-3-Frontend
$ cd ironhack-project-3-frontend
$ npm install
$ npm start
```
You have to create and configure a .env file in the root folder with the following variables:

- REACT_APP_SERVER_URL
- REACT_APP_FRONTEND_URL
- REACT_APP_GOOGLE_KEY
